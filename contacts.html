niftilib contacts

We encourage all concerns, questions and suggestions about the nifti-1 format to be posted to the nifti-1 Message Board (http://nifti.nimh.nih.gov/dfwg/phorum/).

Questions about the nifti_io library should be sent to Bob Cox, through the nifti-1 Message Board.

Questions about the fslio and znzlib libraries should be sent to Mark Jenkinson.

Questions about the niftijlib library should be sent to Kate Fissell (see below).

The project admin for the niftilib sourceforge site is Kate Fissell, contact via <last name> at users.sourceforge.net
