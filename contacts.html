niftilib contacts

We encourage all concerns, questions and suggestions about the nifti-1 format to be posted to the nifti-1 Message Board (http://nifti.nimh.nih.gov/board/).

Questions about the nifti_io library should be sent to Bob Cox and Rick Reynolds, through the nifti-1 Message Board.

Questions about the fslio and znzlib libraries should be sent to Mark Jenkinson, mjenkinson@users.sourceforge.net.

Questions about the niftijlib library should be sent to Kate Fissell (fissell@users.sourceforge.net).

The project admins for the niftilib SourceForge site are:
Kate Fissell fissell@users.sourceforge.net
